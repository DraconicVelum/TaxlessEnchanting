<p align="center">TaxlessEnchanting removes vanilla level scaling<br>
and replaces it with a precise XP-based system for enchanting and anvil usage.<br><br>
It ensures accurate XP consumption<br>
and seamless integration with AuraSkills.<br><br>
Experience enchanting the way it should feel.</p>

---

### <p align="center">Custom XP System</p>
<p align="center">Replaces vanilla level costs with real XP values.<br>
Accurate and consistent progression.</p><br>

### <p align="center">Enchanting Table Support</p>
<p align="center">Fully replaces vanilla XP cost handling.<br>
No flicker, fully synchronized UI.</p><br>

### <p align="center">Anvil Support</p>
<p align="center">Applies XP-based cost to anvil operations.<br>
Displays cost based on actual XP difference.</p><br>

### <p align="center">AuraSkills Integration</p>
<p align="center">Grants Enchanting XP through AuraSkills.<br>
Supports Lucky Table ability.</p><br>

### <p align="center">Config Driven</p>
<p align="center">Enable or disable features easily.<br>
Adjust multipliers and behavior through config.</p>

---

### <p align="center">🧩 Requirements</p>
<p align="center">Paper / Spigot / Bukkit (1.20+ / 1.21 API)</p>

---

### <p align="center">⚙️ Installation</p>
<p align="center">1. Drop the plugin ".jar" into your plugins folder.</p>
<p align="center">2. Restart your server.</p>

---

### <p align="center">⚙️ Configuration Example</p>

```yaml
enchanting:
  enabled: true
  multiplier: 1.0

anvil:
  enabled: true
  multiplier: 1.0

auraskills:
  enabled: true
  lucky:
    enabled: true
  xp:
    enabled: true

general:
  debug: false
```
